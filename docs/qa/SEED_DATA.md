# QA Seed Data

This document defines the reusable QA accounts and seeded records generated by `scripts/seed_qa_data.sh`.

## Prerequisites

- Backend API running at `http://localhost:8000`
- Postgres reachable with:
  - `PGHOST=localhost`
  - `PGUSER=postgres`
  - `PGPASSWORD=postgres`
  - `PGDATABASE=flowtrack_qa`

## Seed Command

From repo root:

```bash
scripts/seed_qa_data.sh
```

The script is idempotent for user and tenant setup. It also ensures tenant memberships required for tenant-scoped FlowTrack APIs.

## Quick DB Validation

```bash
PGPASSWORD=postgres psql -h localhost -U postgres -d flowtrack_qa -At -c \
"select (select count(*) from flowtrack_projects), (select count(*) from flowtrack_tickets), (select count(*) from flowtrack_releases);"
```

Expected result: at least `1|1|1`.
